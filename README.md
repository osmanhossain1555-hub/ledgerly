# Peeduiteun Budget App

Mobile-first budgeting PWA based on the provided UI reference.

## What it does
- Dashboard with spending summary
- Monthly trend chart
- Budgeting overview with donut chart
- Per-category budget progress
- Add, edit, and delete transactions
- Set monthly income
- Update category budgets
- Local storage persistence
- Export and import JSON backup
- Installable PWA with offline support

## Run it
Open `index.html` in a browser, or serve the folder with any static server for best PWA behavior.
