# Finance X Android Project

This is an Android WebView app that wraps a local offline Finance X dashboard.

## What it does
- Opens the Finance X dashboard from local app assets
- Stores transaction data in the WebView's local storage on the device
- Works offline after install
- Phone-first UI

## How to build an APK
1. Install Android Studio on a Windows, Mac, Linux, or Chromebook with Linux support.
2. Open this folder in Android Studio.
3. Let Gradle sync.
4. Go to **Build > Build Bundle(s) / APK(s) > Build APK(s)**.
5. Android Studio will generate a debug APK.

## Important
This package contains the Android project source. It is **not** a compiled APK because the current environment does not include the full Android SDK/Gradle build toolchain needed to compile and sign an APK.
